# Lista Encadeada em Python

Este repositório contém uma implementação simples de uma lista encadeada em Python.

## Descrição

Uma lista encadeada é uma estrutura de dados linear onde cada elemento é um nó que contém um valor e uma referência ao próximo nó na sequência. Esta implementação inclui as operações básicas de adição, remoção e impressão de elementos na lista encadeada.

## Como Usar

1. Clone este repositório para o seu ambiente local.

2. Certifique-se de ter o Python instalado em seu sistema.

3. Execute o código Python `listaencadeada.py`.

4. Digite o nome do arquivo de entrada quando solicitado.

5. O programa lerá o arquivo de entrada e executará as operações especificadas, imprimindo a lista encadeada resultante.

## Exemplo de Saída

Para o arquivo de entrada fornecido, a saída esperada é uma lista encadeada após a adição e remoção de elementos conforme especificado no arquivo de entrada.
